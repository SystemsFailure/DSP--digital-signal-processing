МАТЕМАТИЧЕСКАЯ СТАТИСТИКА


Генеральная совокупность (также называемая популяцией) в математической статистике представляет собой полный набор всех возможных элементов, 
обследуемых в ходе исследования или эксперимента. Например, если исследование проводится для определения среднего роста всех женщин в определенной стране, 
то генеральная совокупность будет состоять из всех женщин, проживающих в этой стране.

Выборка, с другой стороны, представляет собой часть генеральной совокупности, которая выбирается для исследования или эксперимента. 
Выборка является подмножеством генеральной совокупности и должна быть представительной для всей популяции, 
чтобы результаты исследования можно было обобщить на всю генеральную совокупность. Выборка может быть выбрана случайным образом или с использованием других методов выборки, 
таких как стратифицированная выборка или кластерная выборка.

Основной целью использования выборки в математической статистике является получение информации о генеральной совокупности, 
используя только часть данных. Это экономически эффективный способ собрать данные и провести исследование без необходимости охватывать каждый элемент популяции. 
Выборка также позволяет сделать выводы о популяции с определенной степенью достоверности и точности.

Дискретный вариационный ряд – это упорядоченное по возрастанию (как правило) множество вариант  x1, x2, x3
(значений величины X) и соответствующих им частот либо относительных частот.


Математическое ожидание в выборке рассчитывается как среднее арифметическое элементов выборки. Для этого необходимо:
    1. Суммировать все значения элементов выборки.
    2. Разделить сумму на количество элементов в выборке.

Математическое ожидание (M) в выборке можно рассчитать следующей формулой:
    M = (x1 + x2 + ... + xn) / n,
    где x1, x2, ..., xn - элементы выборки, n - количество элементов в выборке.

Пример:
    Пусть у нас есть следующая выборка: 5, 7, 9, 3, 2.
    Математическое ожидание можно рассчитать следующим образом:
    M = (5 + 7 + 9 + 3 + 2) / 5 = 26 / 5 = 5.2.

Таким образом, математическое ожидание в данной выборке равно 5.2.

Обычно частота для выборки обозначается как n, а для генеральной совокупности как N

Относительные частоты расчитываются по формуле:
    w1 = n1 / n - где n = сумма всех n1, n2 и.тд

Основы математической статистики включают в себя следующие понятия:
1. Случайная величина: это величина, значение которой не может быть точно предсказано. Она учитывает случайные факторы.
2. Распределение вероятностей: это функция, которая задает вероятность различных значений случайной величины.
3. Параметры распределения: это числа, которые описывают форму распределения вероятностей, такие как среднее значение, медиана и стандартное отклонение.
4. Выборка: это подмножество значений из генеральной совокупности - полной группы объектов, о которых хотим сделать выводы.
5. Описательная статистика: это методы и показатели, которые используются для описания выборки. Примерами могут быть среднее значение, медиана, дисперсия и корреляция.
6. Интервальные оценки: это методы, которые позволяют оценить неизвестные параметры о совокупности на основе выборки. Например,
   можно построить доверительный интервал для среднего значения выборки.
7. Гипотезы и статистические тесты: это методы, которые позволяют проверить гипотезы о совокупности на основе выборки. 
   Например, можно провести t-тест для проверки разности средних значений двух выборок.
8. Корреляция и регрессия: это методы, которые позволяют исследовать связь между двумя или более переменными. 
   Например, можно построить модель регрессии для предсказания значения одной переменной на основе другой переменной.

Математическая статистика предоставляет инструменты для анализа данных, оценки параметров, проверки гипотез и построения моделей. 
Она является важной дисциплиной в сфере науки о данных и помогает в принятии обоснованных решений на основе статистических выводов.


Мода, медиана и средняя - это три основных показателя, используемых в математической статистике для описания распределения данных.
1. Мода - это значение или значения, которые наиболее часто встречаются в наборе данных. Другими словами, мода - это наиболее популярное значение. 
   Например, в наборе данных {1, 2, 2, 3, 4, 4, 4}, модой будет число 4, потому что оно встречается чаще всего. Набор данных также может иметь несколько мод, 
   если несколько значений встречаются с одинаковой частотой.
   Мода. Мода  дискретного вариационного ряда – это варианта с максимальной частотой. В данном случае.
   Моду легко отыскать по таблице, и ещё легче на полигоне частот – это абсцисса самой высокой точки:

2. Медиана - это значение, которое разделяет набор данных на две равные половины, так что половина значений находится выше медианы, а другая половина - ниже. 
   Для того чтобы найти медиану, сначала нужно упорядочить набор данных по возрастанию или убыванию, а затем выбрать средний элемент. 
   Например, в наборе данных {1, 2, 2, 3, 4, 4, 4}, медианой будет число 3.
3. Средняя - это сумма всех значений в наборе данных, разделенная на количество значений. 
   Средняя, или среднее арифметическое, является наиболее распространенным показателем центральной тенденции. 
   Она может быть вычислена, сложив все значения и поделив на их количество. 
   Например, для набора данных {1, 2, 2, 3, 4, 4, 4}, средняя будет равна (1 + 2 + 2 + 3 + 4 + 4 + 4) / 7 = 20 / 7 = 2,86 (округленно).

Генеральной средней называется среднее арифметическое всех значений этой совокупности:
    x = Σx / N

Выборочной средней называется среднее арифметическое всех значений выборки:
    x = Σx*n / n

РАЗМАХ ВАРИАЦИИ. СРЕДНЕЕ ЛИНЕЙНОЕ ОТКЛОНЕНИЕ. ГЕНЕРАЛЬНАЯ И ВЫБОРОЧНАЯ ДИСПЕРСИЯ


Размах вариации (range of variation) является мерой разброса (дисперсии) данных в статистике. Это разница между наибольшим и наименьшим значением в наборе данных.
Для вычисления размаха вариации необходимо найти наибольшее и наименьшее значения в наборе данных и вычислить их разницу. 
Формула для вычисления размаха вариации:
    Размах = наибольшее значение - наименьшее значение
    Например, у нас есть набор данных: 5, 6, 7, 8, 9. 
    Наибольшее значение = 9
    Наименьшее значение = 5
    Размах вариации = 9 - 5 = 4
    Таким образом, размах вариации для данного набора данных равен 4. 

Размах вариации широко используется в статистике для измерения разброса данных и позволяет определить, 
насколько различны значения в наборе данных. Однако он не учитывает распределение данных и может быть 
подвержен влиянию экстремальных значений.

Среднее линейное отклонение (МЛО) - это мера разброса значений вокруг среднего значения в наборе данных. 
Оно показывает, насколько в среднем каждое значение отклоняется от среднего значения.
МЛО вычисляется следующим образом:
    1. Вычислить среднее значение набора данных.
    2. Для каждого значения набора данных вычислить абсолютное значение разности между значением и средним значением.
    3. Вычислить сумму всех таких разностей.
    4. Поделить сумму на количество значений в наборе данных.

МЛО позволяет определить, насколько велик разброс значений относительно среднего значения. 
Чем больше МЛО, тем больше разброс данных. Он является одним из методов измерения разброса данных, вместе с дисперсией и стандартным отклонением.

В целом - это есть среднее арифметическое абсолютных отклонений всех значений:
    L = Σ|xᵢ - x̄| * nᵢ / n


Выборочная дисперсия  – это среднее арифметическое квадратов отклонений всех вариант выборки от её средней
выборочная дисперсия вычисляется по формуле:
    S² = (Σ(xᵢ - x̄)²) / (n - 1),
    где S² - выборочная дисперсия,
    Σ - сумма всех значений,
    xᵢ - каждое значение из выборки,
    x̄ - среднее значение выборки,
    n - количество элементов в выборке.

Формула для генеральной дисперсии выглядит следующим образом:
    σ² = Σ (xᵢ - μ)² / N
    где:
        - σ² - генеральная (популяционная) дисперсия;
        - Σ - сумма;
        - xᵢ - каждое значение в выборке;
        - μ - среднее значение (математическое ожидание) в выборке;
        - N - размер выборки.

ДИСПЕРСИЯ, КВАДРАТИЧНОЕ ОТКЛОНЕНИЕ
Формула для расчёта дисперсии в математической статистике:
    Дисперсия (Var) вычисляется путём нахождения среднего квадратического отклонения от среднего значения (математического ожидания) элементов выборки. 
    Для выборки с элементами x ᵢ, где i принимает значения от 1 до n, формула для расчёта дисперсии выглядит следующим образом:
        Var = (1/n) * Σ(xᵢ - μ)²,
        где Var - дисперсия выборки,
        n - количество элементов в выборке,
        xᵢ - i-й элемент выборки,
        Σ - сумма всех элементов,
        μ - среднее значение выборки (математическое ожидание).
        
    Таким образом, сначала необходимо вычислить среднее значение элементов выборки, 
    а затем вычислить сумму квадратов разностей каждого элемента от среднего, поделив на количество элементов выборки.

Еще одна формула -  – дисперсия равна разности средней арифметической квадратов всех вариант статистической совокупности и квадрата средней самих этих вариант:
    D = x̄² - m²


Среднее квадратическое отклонение (стандартное отклонение) является одной из основных мер разброса значений в выборке или популяции в математической статистике. 
Оно показывает, насколько значения отклоняются от среднего значения.

Математическая формула для вычисления среднего квадратического отклонения:
    σ = √(Σ(x - μ)² / N)
    - σ обозначает среднее квадратическое отклонение
    - Σ - символ суммы
    - x - значение из выборки или популяции
    - μ - среднее значение выборки или популяции
    - N - количество значений в выборке или популяции

Данная формула вычисляет сумму квадратов разностей каждого значения из выборки или популяции с их средним значением. 
Затем сумма делится на количество значений, 
а затем находится квадратный корень от полученного значения.

Среднее квадратическое отклонение позволяет оценить разброс значений и понять, 
насколько среднее значение является представительным для выборки или популяции.


коэффициент вариации
– это отношение стандартного отклонения к средней, выраженное в процентах:


Коэффициент вариации (CV) - это мера относительной изменчивости или разброса данных, выраженная в процентах. 
Он используется для сравнения дисперсии или стандартного отклонения двух или более наборов данных с разными средними значениями.

Формула для вычисления коэффициента вариации:
    CV = (σ / μ) * 100,
    где:
        CV - коэффициент вариации,
        σ - стандартное отклонение,
        μ - среднее значение.

Математическая статистика - это раздел статистики, который изучает методы и техники сбора, представления, анализа и интерпретации данных, 
а также основные понятия, принципы и теоремы, используемые в статистике.
Подробная информация о математической статистике и коэффициенте вариации доступна 
в специализированной литературе и учебниках по статистике.


АСИМЕТРИЯ И ЭКЦЕСС
Итак, что такое асимметрия и эксцесс? Говоря простым языком, это показатели, характеризующие геометрическую форму распределения. 
Асимметрия характеризует меру скошенности графика влево / вправо, а эксцесс –  меру его высоты.

Данные показатели рассчитываются как для эмпирических, так и для теоретических распределений, которые мы изучили в курсе теории вероятностей, 
и за «эталон» симметрии принято нормальное распределение:
    http://mathprofi.ru/s/asimmetriya_i_excess_clip_image002.jpg

    Очевидно, что любое нормальное распределение строго симметрично относительно своего центра, следовательно, его асимметрия равна нулю. 
    Данный график кажется пологим, но стандартное отклонение  в данном примере достаточно велико (см. на синие точки) и на самом деле такая высота «гармонична». 
    Поэтому эксцесс нормального распределения (любого) принимают за «отправную» нулевую точку.

Для расчета асимметрии используют формулу;
    A = x̄ - M₀ / σ - где σ - среднее квадратичное отклонение
    Что тоже логично, ведь у разных распределений – разный «разброс» значений и разные представления о мере асимметрии.

Коэффициент экцесса расчитывается так:
    E = m4/σ - 3 - m4 - Центральный эмпирический момент 4 порядка, σ - ст.отклонение
    m4 = Σ(xᵢ - x̄ᵢ)⁴ / n - для не сгруппированных данных, и
    m4 = Σ(xᵢ - x̄ᵢ)⁴ * nᵢ / n - для сформированного вариационного ряда

Коэффициент эксцесса эмпирического распределения (kurtosis) может быть вычислен с использованием следующей формулы:
    kurtosis = (n*(n+1)/(n-1)(n-2)(n-3)) * (Σ((xᵢ-μ)/σ)⁴) - 3*((n-1)²)/((n-2)(n-3))
    где:
        - n - объем выборки
        - xᵢ - значения выборки
        - μ - среднее значение выборки
        - σ - стандартное отклонение выборки
        - Σ - сумма всех значений от i=1 до n

Заметьте, что данная формула используется для вычисления эксцесса эмпирического распределения на основе выборки, а не для теоретического распределения.
