Метод наименьших квадратов (МНК) - это статистический метод аппроксимации данных, используемый для нахождения наилучшей подгонки к заданному набору точек. 
Формула МНК позволяет минимизировать сумму квадратов расстояний между реальными и предсказанными значениями.

Формула МНК для одномерного случая выглядит следующим образом:
    y = mx + b,
    где y - зависимая переменная, x - независимая переменная, m - коэффициент наклона (наклон прямой), b - свободный член (точка пересечения с осью y).

Основная идея формулы МНК состоит в том, что мы стремимся найти значения m и b, которые минимизируют сумму квадратов разностей между каждым реальным значением 
y и предсказанным значением на основе линейной модели. Сумма квадратов разностей в этой формуле является функцией ошибки, которую мы пытаемся минимизировать.

Чтобы найти оптимальные значения m и b, мы можем использовать различные численные методы оптимизации, такие как градиентный спуск или матричные методы.

Общая формула МНК для многомерного случая (когда у нас есть несколько независимых переменных) имеет вид:
    Y = Xβ + ε,
    где Y - вектор зависимых переменных, X - матрица независимых переменных, β - вектор коэффициентов регрессии (множество коэффициентов наклона), ε - вектор случайных ошибок.

В этом случае, чтобы найти оптимальные значения β, мы можем решить систему уравнений методом наименьших квадратов, используя методы линейной алгебры, 
такие как обратная матрица или метод наименьших квадратов через QR-разложение.

Метод наименьших квадратов широко применяется в различных областях, таких как экономика, физика, 
инженерия и социальные науки, для аппроксимации данных и предсказания значений на основе статистической модели.


Формула МНК через минимум квадратов отклонения имеет вид:
    E = Σ(xᵢ - f(xᵢ))² -> min
    где:
        xᵢ - наблюдаемое значение,
        f(xᵢ) - предсказанное значение,
        n - количество наблюдений.


Метод наименьших квадратов (МНК) используется для нахождения наилучшей линейной аппроксимации данных, 
которые подчиняются линейной зависимости. Формула для линейной зависимости, удовлетворяющей МНК, имеет вид:
    y = a * x + b,
    где y - зависимая переменная,
    x - независимая переменная,
    a - коэффициент наклона (угловой коэффициент) линии,
    b - коэффициент смещения (пересечения с осью y) линии.

Метод наименьших квадратов минимизирует сумму квадратов отклонений между фактическими значениями зависимой переменной и значениями, 
предсказанными моделью (аппроксимацией). Формула для нахождения коэффициентов a и b через МНК имеет следующий вид:
    a = (n * ∑(x*y) - ∑x * ∑y) / (n * ∑(x²) - (∑x)²)
    b = (∑y - a * ∑x) / n,
    где n - количество наблюдений,
    ∑ - сумма всех значений (например, ∑(x*y) - сумма произведений x и y).

Эти формулы находят коэффициенты a и b, которые оптимально аппроксимируют данные в линейной зависимости.
